# Image Resizer

# Frontend
## Development setup
```
npm install
npm run serve
```

# API
## Development setup
```
npm i -g serverless
npm i -g serverless-offline
npm install

sls offline
```

## Deployment
1. Download and install [AWSCLI](https://aws.amazon.com/cli/)
2. run ```aws configure``` and input your key and secret key
3. run ```npm run deploy```
